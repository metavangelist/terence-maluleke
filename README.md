# Terence Maluleke

Artist website for Terence Ntsako Maluleke — digital artist and painter.

## Live

https://maluleke.art

## Local preview

```bash
npm run dev
```

- **Desktop:** http://localhost:5501
- **Phone (same Wi‑Fi):** use the `http://<your-mac-ip>:5501/` URL printed by `npm run dev` — do **not** use `localhost` on your phone.

Alternative (Python):

```bash
python3 -m http.server 5501 --bind 0.0.0.0
```

## Structure

Single-page site with vertical scroll sections: Home, Info, Paintings, Prints, Assamblage, Study, Calendar.

CMS: Sanity Studio in `studio/` (https://terence-art.sanity.studio).
